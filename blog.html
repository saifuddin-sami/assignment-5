Important FAQ

What is the Document Object Modes (DOMS)?

The Document Object Model (DOM) is a hierarchical representation of a web document that allows programming languages, like JavaScript, to interact with and manipulate the content and structure of the document dynamically. It treats each part of the document as an object, enabling tasks like changing text, styles, and handling events.

How do you select an element from the DOM?

You can select an element from the DOM using methods like getElementById(id) for a specific ID, querySelector(selector) for a CSS selector, and getElementsByClassName(className) for elements with a specific class. These methods allow you to access and manipulate elements easily in JavaScript.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element instead of multiple listeners on child elements. This allows the parent to handle events for its children, even if they are added dynamically later.

How do you manipulate an element's attributes and styles using the DOM?

I can manipulate an element's attributes using `setAttribute()` to change or add attributes, and `getAttribute()` to retrieve them. For styles, use `element.style` to modify CSS properties, like `element.style.color = 'red'` to change the text color.
